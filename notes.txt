  whole app logic: 
  1. User Input
  2. useUserSync.js run to send user data to backend to sync user
  2. goes to api.js functions (functions hold data and send data to middleware at index.ts [exp: app.use("/api/comments", commentRoutes);])
  3. middleware calls the routes in routes folder, then calls the controller
  4. controller functions handle the logic and process everything


  hook = functions with timing on what should happen when what event triggers it
  exp: 
  1. when user sign in, isSignedIn = true & user = {email, name, image}
  2. useEffect inside useUserSync runs because [isSignedIn, user] changed 
  3. then hook sends request to POST /api/users/sync
